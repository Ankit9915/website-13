Plugin for Sketch

Automatically compresses images exported from Sketch with ImageOptim.

To install the plugin:

  1. Download the plugin
  2. Unarchive ImageOptim.sketchplugin.zip
  3. Open ImageOptim.sketchplugin

The plugin requires ImageOptim.app to be installed.

This plugin is Free and Open-Source.
